# Library

## Javascript
### Mermaid
text를 graph형태로 표현해 주는 library

### Docsify
Markdown을 SPA 형태로 보여주는 JavaScript

## DB Migration
### Flyway
파일명에 지정된 번호 순서로 migration script를 실행

### Liquibase
changeset에 지정된 author + id를 기반으로 migration script를 실행

실행 순서가 지정되지 않음
